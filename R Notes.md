### Data Type
Common Types  
logical: TRUE/FALSE (or T/F)  
integer: 1L  
double (numeric): 1.23  
character: "1.23"  
Convert data types:  
x <- as.numeric("1.23") 


### Vector & Matrix
x <- c(1, 2, 4, 7)

Furthermore, you can assign names to a vector’s elements and refer to each element by its name. Such vectors are called “named vectors”.  
names(x) <- c("a", "b", "c", "d")
